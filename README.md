# JEECMS模板 #

> **template-gjyy.zip** 是一个高效课程专题站点模板。
> 
> **template-syfz.zip** 是一个中学站点模板。
> 
> 以上模板基于JEECMS V9.3（http://bbs.jeecms.com/res_base/jeecms_com_bbs/upload/201803/jeecmsv9.3-ROOT.zip）开发。

# 版本号命名规则 #

X: 版本号, Y: 修订号, Z: 迭代次数

> 如：Ver.X.Y.Z 代表第X版，第Y次修订，第Z次迭代。
